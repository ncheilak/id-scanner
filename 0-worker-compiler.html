CSV Format Guide

Your CSV file must have a header row with these column names (order doesn't matter, extra columns are ignored):

firstName,lastName,trade,company,phone,workerId,ecName,ecPhone,natId,certExp

Only firstName, lastName, and trade are required. All other columns are optional.

Example row:
John,Doe,Electrician,ABC Corp,+1 555 1234,W-1001,Jane Doe,+1 555 9999,XX-0001,2026-12-31

Trade must be one of: Foreman, Carpenter, Electrician, Plumber, Concrete Worker, Steel Fixer, Crane Operator, Scaffolder, Laborer, Safety Officer, Site Engineer, Surveyor, Welder, Driver, Other
